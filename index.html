About me

I love writing (clean) code. Since the day I wrote my first program in Java and shaped it to produce the desired output, back then it felt almost like magic (and still does), being able to get the computer to do whatever I wanted. A spark ignited in my brain that opened my eyes to the endless possibilities of using software to solve practical problems in real life.

Software engineering is like a never-ending puzzle in which I am fervidly committed to solving. These puzzles can be sometimes very complex and challenging. But that is exactly what makes me tick. Coming up with a solution, test your solution (in a separate Git branch, of course, 😉) successfully against all possible scenarios, and then seeing your code running smoothly on production is oddly satisfying.

This passion and thirst for knowledge carried me through my career. It is where I was fortunate enough to work for numerous clients in many projects with different technology stacks (Java, Android, C#.Net, Azure DevOps, CI/CD). Throughout my career, I built up specific knowledge in various industries and gained a lot of experience. I am grateful to use and share these skills/knowledge in my everyday work.

My specialties include problem-solving, quickly learning new skills and programming languages, SOLID principle, software design patterns, code optimizations, and coaching fellow team members.
